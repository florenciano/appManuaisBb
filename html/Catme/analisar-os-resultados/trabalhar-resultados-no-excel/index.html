Trabalhar resultados no Excel

Caso seja necessário trabalhar sobre os dados brutos do Catme para realização de outros cálculos ou mesmo para geração de gráficos, as tabelas poderão ser exportadas para uma planilha Excel.

Sumário

Escolha o local em que o relatório será salvo. Será gerado um arquivo no formato CSV.

Abra o arquivo, selecione a primeira coluna e, na aba Dados, escolha a opção Texto para colunas.

Siga os passos indicados e, ao finalizar, os dados estarão disponíveis para análise no Excel.

Outra opção é copiados os dados do Catme e colá-los diretamente em uma planilha Excel.

A seleção deve começar um parágrafo antes do início da tabela.
