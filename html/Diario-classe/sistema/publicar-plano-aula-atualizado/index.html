Publicar plano de aula atualizado

Sumário

Após a publicação do Plano de Aula, qualquer alteração necessária deve refletir no Plano de Aula Atualizado. Para isso, o sistema copia automaticamente os dados dos Planos de Aula publicados para o Plano de Aula Atualizado, facilitando o trabalho do docente na atualização do programa quando necessário.

Você pode utilizar os filtros (curso, ano, período, disciplina e turma) para encontrar o plano de aula desejado.

Somente os planos das turmas vinculados ao docente, referentes ao período letivo corrente, serão exibidos para preenchimento. Todos os documentos terão inicialmente o status “Preenchimento pendente”.

No caso de cancelamento e reposição de aula, as alterações de data são refletidas automaticamente no Plano de Aula Atualizado, sem haver perda dos conteúdos inseridos anteriormente.

Somente após o preenchimento de todos os campos obrigatórios do Plano de Aula, os botões “Salvar” e “Salvar e visualizar” serão habilitados.

Salva as informações e permanece na tela do plano de aula.

Salva e gera um documento PDF idêntico à versão disponibilizada aos alunos no Blackboard.

Atenção: o bloqueador de pop-up precisa estar desativado

Ao salvar o Plano de Aula Atualizado ele é automaticamente publicado, portanto o status do documento permanece como “Publicado” e a versão mais recente é disponibilizada para os alunos.

Caso tenha replicado esse plano para outra disciplina será necessário replica-lo novamente. Isso não é feito automaticamente.
