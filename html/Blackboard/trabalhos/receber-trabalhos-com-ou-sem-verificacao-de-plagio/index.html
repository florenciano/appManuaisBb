Receber trabalhos com ou sem verificação de plágio (Exercício)

A ferramenta Exercício permite receber trabalhos dos alunos diretamente pelo ambiente, evitando o recebimento por e-mail. Dentre seus benefícios, destacam-se:

  • Recebimento de trabalhos individuais ou em grupo, com controle das versões entregues.
  • Controle dos prazos de entrega: trabalhos recebidos no prazo, atrasados e em aberto.
  • Rastreamento de plágio: por meio da comparação do trabalho com outros documentos da base de dados do Insper, da Internet e de outras instituições de ensino que usam Blackboard. Para verificação de plágio, só são aceitos arquivos nos formatos doc, docx, ppt, pptx, odt, txt, pdf, rtf e html.

Desde a atualização do Blackboard de janeiro/2015, a ferramenta SafeAssignment, até então usada para rastreamento de plágio, foi agregada à ferramenta Exercício.

As antigas atividades criadas com o SafeAssignment continuam acessíveis no ambiente das disciplinas, sem qualquer alteração.

Sumário

Criar Exercício

Defina o nome do exercício e digite as instruções para os alunos na caixa Instruções.

Se necessário, anexe um ou mais arquivos. Essa opção é interessante quando os alunos tiverem que seguir algum modelo de documento, ou quando houver alguma leitura prévia ou material de consulta.

Defina a data de vencimento.

Os exercícios enviados após a data de vencimento serão aceitos, mas ficarão indicados como Atrasado. Para não receber exercícios atrasados, limite a disponibilidade marcando a opção Exibir até.

Defina o valor máximo de pontos possíveis (em geral, 10) e, se necessário, associe uma Rubric.

Indique se a entrega do exercício será individual ou em grupo. Caso seja em grupo, mova-os para a lista da direita.

Atenção: Caso a atividade seja em grupo, os mesmos devem ser criados previamente.

Defina quantas vezes o exercício poderá ser enviado. No caso de mais de uma tentativa, escolha qual delas será exibida na Central de Notas e, consequentemente, qual será considerada no cálculo da nota.

Escolha se haverá verificação de plágio nos envios e, em caso positivo, se os alunos terão acesso ao relatório.

Caso não queira que os exercícios recebidos entrem para a base de comparação do Insper, marque a opção de rascunho.

Ative essa opção para ocultar os nomes dos alunos durante o período de avaliação, tornando-os anônimos.

Se necessário, delegue a correção da atividade a outros avaliadores. Eles poderão alterar as avaliações realizadas por outro avaliador, mas os instrutores possuem habilidade de revisão.

Só serão listados como possíveis avaliadores usuários com o perfil de Professor e Auxiliar de Ensino. Não é possível delegar a avaliação a outros alunos.

Por padrão, o formato das notas é Pontuação. Exibições secundárias não são visualizadas pelos alunos.

Defina se o exercício ficará ou não disponível e por quanto tempo.

Caso queira programar a atividade para ficar disponível aos alunos somente a partir de uma determinada data, marque a opção Tornar o exercício disponível e configure a opção Exibir depois com a futura data de disponibilização.

Como funciona a verificação de plágio

  1. Professor cria um Exercício no ambiente da disciplina e marca a opção o Rastrear possibilidades de plágio via SafeAssign.
  2. Aluno submete o trabalho.
    • Caso o professor não tenha marcado a opção “Considerar envios como rascunhos” no momento de criação da atividade, o trabalho do aluno é automaticamente adicionado à base da instituição. Uma vez enviado, não é possível remover o trabalho da base de dados.
    • Caso o aluno marque a opção “Concordo em enviar meus relatórios para o banco de dados de referência global”, o trabalho é adicionado a uma base global compartilhada por todos os clientes da Blackboard. Uma vez enviado, não é possível remover o trabalho da base de dados.
  3. O trabalho do aluno é comparado com outros trabalhos da base da instituição, com a base global e com resultados de busca da internet.
  4. A partir das comparações é gerado um relatório que indica similaridades entre o trabalho enviado pelo aluno e as bases de comparação disponíveis.
    • Caso o professor tenha marcado a opção “Permitir que os alunos visualizem o relatório de plágio” no momento de criação da atividade, o aluno também tem acesso ao relatório.
