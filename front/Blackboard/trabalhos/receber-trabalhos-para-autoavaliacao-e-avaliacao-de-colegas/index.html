Receber trabalhos para autoavaliação e avaliação de colegas

A ferramenta Autoavaliação e avaliação de colegas possibilita que os alunos submetam trabalhos e, de acordo com critérios predefinidos, avaliem os trabalhos um dos outros. A avaliação pode ser anônima ou identificada.

Sumário

Funcionamento da ferramenta

Uma vez disponível aos alunos, a atividade possui dois grandes momentos: primeiro, os alunos submetem seus trabalhos; na sequência, ocorre o período de avaliação. Acompanhe o fluxo complexo na ilustração.

Restrições da ferramenta

Antes de usar a ferramenta, é necessário ter ciência de algumas restrições:

Só é possível criar atividades que envolvam a turma toda.

Não é possível criar atividades envolvendo apenas alguns alunos ou mesmo grupos específicos de alunos. No período de avaliação, o sistema faz o sorteio e distribuição dos trabalhos para correção considerando todos os alunos inscritos na disciplina.

Se algum aluno não submeter o trabalho no período de entrega, ainda assim entrará no sorteio e distribuição de trabalhos para correção.

Pode acontecer de alunos receberem para corrigir o trabalho “vazio” de quem não fez. Além disso, quem não fez a atividade receberá trabalhos para corrigir e, se não o fizer, deixará os colegas que seriam avaliados sem um dos feedbacks.

Não é possível modificar os prazos após sorteio e distribuição dos trabalhos.

Uma vez que o sistema sorteou e distribuiu os trabalhos para correção, não é possível alterar o prazo de entrega. Assim, se os alunos não tiverem entregue o trabalho no prazo, não terão uma segunda chance. Apesar disso, os trabalhos não entregues entrarão no sorteio para que os colegas o avaliem com nota zero.

Os critérios de avaliação permitem nota parcial, mas não admitem valores decimais.

O sistema só admite atribuição de notas inteiras na avaliação. Por exemplo: se determinado critério vale até 2 pontos e admite nota parcial, os alunos poderão atribuir as notas 0, 1 ou 2, mas não poderão atribuir notas quebradas, como 1,5.

Os trabalhos submetidos não passam por verificação de plágio.

Para verificar eventuais plágios nos trabalhos enviados pelos alunos é necessário analisar os arquivos no SafeAssign.

Criar atividade

Informe nome, instruções e período em que os alunos poderão submeter seus trabalhos.

Informe período em que os alunos farão a avaliação de trabalhos dos colegas.

Se “sim”, aluno não sabe de quem é o trabalho que recebeu para avaliar, nem quem o avaliou.

Se “sim”, alunos também avaliarão seus próprios trabalhos.

Se “sim”, após validação do professor, alunos recebem as notas/feedbacks de seus pares.

Quantos trabalhos cada aluno receberá para avaliar, (excluindo o seu próprio, em caso de autoavaliação).

Configure as regras de disponibilização da atividade para os alunos e confirme.

Criar pergunta

Uma vez que a atividade é criada é preciso criar uma ou mais perguntas para que os alunos desenvolvam a atividade.

Cada pergunta equivale a uma “entrega” que o aluno deve fazer, seja em forma de texto digitado no próprio ambiente, seja em forma de arquivo anexo.

Se você deseja que os alunos entreguem um único arquivo, basta criar uma única pergunta, mesmo que o enunciado possua mais de um tópico. Se você deseja que os alunos apresentem diversas respostas, é necessário criar mais de uma pergunta.

Em Texto da pergunta, insira o enunciado que os alunos enxergarão no momento que estiverem submetendo a resposta/arquivo.

Em Resposta do modelo, insira a resposta esperada/considerada correta.

Os alunos a enxergarão apenas no momento em que estiverem corrigindo atividades.

O campo Resposta do modelo não é obrigatório. A depender do caso, você pode deixá-lo vazio, explicitando nos passos seguintes os critérios de avaliação.

Adicionar critérios de correção

Após criar uma ou mais perguntas é necessário informar seu critério de correção. É com base nos critérios que os alunos realizarão a autoavaliação e avaliação dos colegas.

Ao clicar em Critérios, você poderá criar um novo critério.

Se preferir, você também pode criar critério de contagem de palavras A ou reaproveitar modelos de critérios B.

Descreva o critério.

Informe o total de pontos e escolha o modelo de atribuição:

  • Tudo ou nada: não admite notas parciais. Ex.: se pontos possíveis for 2, aluno só pode atribuir 0 ou 2.
  • Crédito parcial: admite notas parciais (não decimais). Ex.: se pontos possíveis for 2, aluno pode atribuir 0, 1 ou 2.
