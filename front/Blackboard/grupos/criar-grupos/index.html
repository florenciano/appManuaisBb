Criar grupos

Há diferentes possibilidades de criação de grupos de alunos no Blackboard.

Sumário

Inscrição automática

Escolha Inscrição automática para que os próprios alunos ingressem em um grupo.

Escolha um nome para identificar o conjunto de grupos que será criado.

Defina quais ferramentas de colaboração ficarão à disposição do grupo para troca de informações e comunicação entre os membros.

Defina o nome da página em que os alunos deverão fazer suas inscrições nos grupos.

Marque a opção Mostrar membros para que os alunos saibam quem já está inscrito nos grupos correspondentes.

Defina quantos grupos deverão ser criados. Os nomes serão compostos pelo nome do grupo + numeração sequencial (veja o exemplo abaixo).

Para que os alunos possam se inscrever nos grupos criados é necessário disponibilizar a página de registro em alguma área do ambiente, como por exemplo, a área Conteúdos.

Na visão de aluno, ao clicar no link de grupos serão exibidos todos os grupos da disciplina e a opção para que ele se inscreva em cada um deles.

Ao se registrar em um grupo, o aluno visualiza os membros daquele grupo, as ferramentas disponíveis para interação e também as atribuições (exercícios) a serem ser desenvolvidas.

Inscrição manual

Escolha Inscrição manual para determinar quais alunos comporão cada grupo.

Escolha um nome para identificar o conjunto de grupos que será criado.

Defina quais ferramentas de colaboração ficarão à disposição do grupo para troca de informações e comunicação entre os membros.

Selecione na lista da esquerda, os alunos que farão parte do grupo. Repita o procedimento para todos os grupos criados.

Inscrição aleatória

Escolha Inscrição manual para determinar quais alunos comporão cada grupo.

Escolha um nome para identificar o conjunto de grupos que será criado.

Defina quais ferramentas de colaboração ficarão à disposição do grupo para troca de informações e comunicação entre os membros.

Configure as opções para criação aleatória dos grupos.
