Blackboard > Criar blog

Criar blog

É possível disponibilizar ao aluno o acesso à ferramenta Blog, que pode conter um ou mais blogs, ou disponibilizar o link direto a um blog específico.

Para inserir a ferramenta de Blog em uma área de conteúdo da disciplina.

Sumário

A página geral dá acesso a todos os blogs criados na disciplina.

Após selecionar essa opção, defina o nome do link e as configurações de disponibilidade.

É criado um link geral para todos os blogs existentes na disciplina.

Uma vez que a ferramenta já está disponível é preciso criar um blog.

Para inserir um blog específico na área de conteúdo da disciplina:

Primeiramente é necessário criar o blog e, na sequência, selecioná-lo na lista.

Para criar um blog:

Defina o nome do blog e escreva as instruções.

Se necessário, configure as restrições de data/hora.

Defina se será criado um único blog em que todos os alunos poderão contribuir ou se cada aluno terá um blog individual.

Caso o blog seja avaliado, a participação dos alunos pode ser acompanhada na Central de notas.

Caso esteja inserindo um blog específico na área de conteúdo da disciplina:

Selecione o blog na lista e, na tela seguinte, defina o nome do link e as configurações de disponibilidade.

Ao entrar no Blog serão exibidas informações como total de postagens, total de comentários e usuários que participaram.

Para criar um Blog específico para um Grupo de alunos, a opção Blogs deve ser marcada no momento de criação do grupo. Somente os alunos do grupo terão permissão para visualizar e participar do blog.

Para acompanhar a participação dos grupos nos blogs, basta selecionar a opção Blogs no menu Ferramentas do curso.

Ao disponibilizar a ferramenta de Blog a um grupo ou conjunto de grupos é criado um blog específico para cada grupo.
