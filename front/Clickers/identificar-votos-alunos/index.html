Identificar os votos dos alunos

Sumário

Criar lista de alunos

Para identificar os responsáveis pelos votos é necessário preparar uma listagem de alunos e carregá-la no TurningPoint. Uma maneira simples de obter a listagem é no ambiente online da turma.

Acesse a Central de Notas do ambiente online da disciplina para acessar o nome dos alunos.

Copie os dados das colunas "Nome" e "Sobrenome" da planilha gerada pelo Blackboard.

Cole os dados na planilha fornecida pelo Help Desk, que já contém o código de identificação de cada aparelho.

Após a criação da lista de alunos, é necessário vincular a lista à apresentação do PowerPoint.

Vincular lista de alunos aos slides

Antes de tudo, abra o PowerPoint com o plug-in do TurningPoint. Caso já tenha preparado sua apresentação de slides, localize o arquivo.

Há duas maneiras de passar a lista de alunos criada no Excel para o TurningPoint.

  1. Copiar os dados do Excel e colar no TurningPoint.

Primeiramente, ordene as colunas do TurningPoint na seguinte ordem (clicando e arrastando):

  1. Device ID(s)
  2. User ID
  3. First Name
  4. Last Name

Copie as quatro colunas da planilha Excel.

Cole os dados no TurningPoint.

No slide, selecione a lista criada.

  1. Salvar a planilha Excel com a lista de alunos em formato CSV e importá-la no TurningPoint.

Atenção!

Antes de salvar, confira se a lista foi criada corretamente, sem duplicidade de colunas.

No slide, selecione a lista criada.
