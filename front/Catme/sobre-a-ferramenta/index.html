Sobre a ferramenta

Sumário

O que é a ferramenta Catme

Catme Smarter Teamwork é uma ferramenta que permite realizar pesquisas de avaliação entre pares.

Concebida em 2003 por pesquisadores de diferentes instituições norte-americanas, atualmente o Catme é utilizado em mais de 600 instituições em 48 países.

Acesse info.catme.org

A ferramenta possibilita:

  • Coletar informações dos alunos e fornecer feedback individualizado.
  • Entender a dinâmica de funcionamento das equipes, analisando a contribuição de cada aluno para o andamento do time.
  • Tomar ciência de possíveis problemas que estejam ocorrendo nas equipes de trabalho.

O vídeo Overview of CATME Team-Maker System mostra o passo a passo para criação de uma pesquisa.

Duração: 14min

Para informações detalhadas sobre o funcionamento do Catme recomendamos a leitura da página de ajuda (para acessar é preciso criar uma conta e fazer o login).

Em 2013 o Catme passou a ser adotado na Graduação para avaliação da competência "trabalho em equipe". A ferramenta, no entanto, está disponível a todos os professores interessados.

Funcionamento do Catme

O Catme trabalha com uma estrutura hierárquica básica de dois elementos: Class e Activity

Primeiramente é necessário criarmos uma Class, categoria maior dentro da qual criamos uma ou mais Activities, que são as pesquisas de avaliação de pares

Ao criarmos uma pesquisa devemos informar quem serão os alunos participantes. Normalmente eles correspondem aos alunos de uma mesma turma/disciplina.

Ao associar os alunos a uma determinada pesquisa é possível usar dois critérios para informar quais são suas equipes:

  • Team-Maker: as equipes são definidas automaticamente com base em critérios demográficos.
  • CATME BARS: as equipes são informadas ao Catme juntamente com a listagem de alunos. Esta deve ser a opção escolhida.

Embora os alunos sejam associados a uma pesquisa em específico, é possível reaproveitá-los de uma pesquisa para outra.

Criar conta

Preencha seus dados pessoais, informando “Insper” no campo Institution e seu e-mail do Insper no campo Email Address.

Após análise das informações pelos administradores do Catme um e-mail será enviado para finalização do cadastro.

Acesse a mensagem enviada para o e-mail cadastrado, clique no link indicado e crie uma senha com no mínimo 6 caracteres.

Com a conta criada, acesse o link info.catme.org e clique em Login to Existing Account.
