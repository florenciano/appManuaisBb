Detalhes geral do Projeto

Nome do Projeto
Manuais Blackboard

Diretório de gravação do Projeto
Material armazenado no computador no seguinte endereço*: D:\wamp\www\projetos\appManuaisBb\front
* diretório default de desenvolvimento de projetos. Segue a diretriz de endereço recomendado e gerenciada pelo Wamp (plataforma de desenvolvimento de aplicação web).

Objetivo do Projeto
Transpor os manuais do Blackboard da versão .pdf para .html mantendo a mesma (ou a mais próxima de) experiência de uso do modelo de origem.

Requisitos mínimos do Projeto
Responsive web design (RWD); Cross-browser compatível com todos os browser que recebem suporte do Blackboard (veja lista).

Metas do Projeto

  • Permitir que o usuário tira as principais dúvidas referente ao Blackboard por meio da navegação das páginas do projeto;
  • Facilitar a busca por determinados assuntos de interesses;
  • Poder imprimir o documento mantendo a mesma experiência do formato original .pdf;

Início de desenvolvimento do Projeto
dezembro de 2013

Estrutura do Projeto

O projeto esta sob a seguintes estrutura de diretórios:

Diretrizes da estrutura de diretório

A estrutura de diretórios esta segue as recomendações de boas práticas para o desenvolvimento de projetos web. Esta estrutura esta baseado num dos mais popular templantes boilerplate disponível para desenvolvimento front-end. A seguir a função de cada diretório desta estrutura:

  • Diretório raiz (front)

    Diretório responsável para armazenar o arquivo índice do diretório (index.html) e as demais, quando houver, páginas HTML. Os subs-diretórios também ficarão alocados aqui. Ainda sobre este diretório fica reservado a alocação de outros arquivos que complementam o desenvolvimento necessário do projeto podendo aumentar esta lista conforme necessidade.

    • 404.html
    • humans.txt
    • robots.txt
    • favicon.ico
  • Diretório de imagens (img)

    Ficarão alocados neste diretórios todas as imagens necessárias para o desenvolvimento do projeto.

    Diretório de imagens blackboard
    Local de armazenamento das imagens do manual. Seus subdiretórios podem aumentar conforme a expansão no desenvolvimento do projeto.

    Diretório de imagens ui
    Local exclusivo de armazenamento das imagens da GUI(Graphical User Interface).

  • Diretório de estilos (sass)

    Neste diretório ficarão alocados os arquivos .sass que gerarão os documentos .css do projeto. O diretório plugins contém os recursos adicionais de estilos, podendo ser adicionados quantos forem necessários.
